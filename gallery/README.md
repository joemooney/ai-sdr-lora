# R4W Signal Gallery

This directory contains visualizations of various waveforms, channel effects, and performance curves generated by R4W.

## Directory Structure

- **waveforms/** - Constellation diagrams and spectrum plots for various modulation schemes
- **channels/** - Visualizations of channel effects (AWGN, fading, multipath, Doppler)
- **ber_curves/** - Bit Error Rate vs SNR curves for different waveforms

## Generating Images

To regenerate all gallery images:

```bash
cargo run --example gallery_generate
```

Or use the Makefile targets:

```bash
make gallery           # Generate all images
make gallery-clean     # Remove all generated images
make gallery-waveforms # Generate waveform images only
make gallery-channels  # Generate channel effect images only
```

## Waveform Gallery

### Phase Shift Keying (PSK)
- BPSK - Binary PSK (1 bit/symbol)
- QPSK - Quadrature PSK (2 bits/symbol)
- 8PSK - 8-Phase PSK (3 bits/symbol)

### Quadrature Amplitude Modulation (QAM)
- 16QAM - 16-point QAM (4 bits/symbol)
- 64QAM - 64-point QAM (6 bits/symbol)

### Frequency Shift Keying (FSK)
- 2FSK - Binary FSK
- 4FSK - 4-level FSK

### LoRa CSS Modulation
- SF7-SF12 chirp spread spectrum
- Waterfall spectrograms showing chirps

## Channel Effects Gallery

### AWGN (Additive White Gaussian Noise)
- Various SNR levels: 0dB, 5dB, 10dB, 20dB

### Rayleigh Fading
- Flat fading constellation distortion

### Rician Fading
- With K-factors of 1, 5, 10

### Multipath (TDL)
- EPA, EVA, ETU channel profiles
- Frequency-selective fading

### Doppler
- Jake's spectrum at various speeds
- Time-varying fading

## BER Curves

- PSK comparison (BPSK, QPSK, 8PSK)
- QAM comparison (16QAM, 64QAM)
- LoRa SF comparison (SF7-SF12)
